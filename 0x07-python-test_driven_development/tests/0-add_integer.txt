The 0-add_integer module
======================

Adds two integers or two floats. A float will be casted to an int if passed
as an argument

Testing import
>>> add_integer = __import__('0-add_integer').add_integer

Add two integers
>>> add_integer(1, 2)
3

Add two negative integers
>>> add_integer(-1, -2)
-3

Add two negative floats
>>> add_integer(-1.25, -2.25)
-3

Add two floats
>>> add_integer(1.50, 2.50)
3

Add int to float
>>> add_integer(1, 2.25)
3

Add float to int
>>> add_integer(1.25, 2)
3

Add negative int to float
>>> add_integer(-1, 2.50)
1

Add negative float to int
>>> add_integer(-2.50, 1)
-1

None as an argument
